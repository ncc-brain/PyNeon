# PyNeon
PyNeon is a light-weight package to work with Neon (Pupil Labs) multi-modal eye-tracking data.

## Documentation
[PyNeon Documentation](https://ncc-brain.github.io/PyNeon/) contains public API and tutorials as Jupyter notebooks.
